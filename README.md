# Calculator (React + Redux) ![ci status](https://travis-ci.org/dontry/calculator.svg?branch=master)

This project is built with React + Redux + CSS-grid

## Install the program

```bash
yarn install
```

## Launch the program

```bash
yarn start
```

## Unit Test

Powered by Jest

```bash
yarn test
```

### Integration Test

Powered by Cypress

```bash
yarn run cypress
```
